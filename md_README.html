About The Project

My project is to create a chess game in C. Players can start a game and play by indicating the pieces to be moved using Forsyth-Edwards Notation (FEN), while respecting the rules for moving pieces, as well as the special moves and different possible endgames. If they wish, the game can be saved and loaded.

Over time, interesting additions may be made to improve this project.

Getting Started

Installation

  1. Clone the repo
    git clone https://github.com/EnzoSergiani/Chess.git
  2. Move to the root
    cd Chess/
  3. Build the project
    make;
  4. Run it
    ./bin/main
